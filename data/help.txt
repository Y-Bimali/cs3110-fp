Supported commands are listed below. Commands are case insensitive.
Tableau and foundation columns are 1-indexed from left to right.
          
Rules: Show rules.

Help/Commands: Show commands.

Count: Show count of moves made so far.

New Game: Start a new game with draw 1.

New Game 3: Start a new game with draw 3.

Autowin: If possible, autocomplete the game (the stockwaste must be empty and
  all tableau cards must be visible.)

Quit: Quit program.

Theme <theme name>: Change color theme of game to <theme name>. Supported themes
  include Classic, Spaceship, Under the Sea, and April Fools

Draw/D: Draw a card from the stock to the waste, or return all cards in 
the waste back to the stock.

S to F: Move the stock card to the foundation.

S to T#: Move the stock card to the tableau at index #.

T# to F: Move a tableau card at index # to the right index in foundation.

F to T#: Move a foundation card to the tableau at index #.

T# to T#: Move tableau card(s) from column (1st #) to column (2nd #).
